## 🚀 使用建議

### 1. 測試流程
```bash
# 啟動應用
python3 app.py

# 訪問網站
http://localhost:5000

# 初始化數據
http://localhost:5000/init-db

# 測試流程
1. 註冊/登入
2. 瀏覽團購（查看新設計）
3. 查看團購詳情
4. 嘗試跟團（看看表單）
5. 開新團購
6. 查看我的訂單
7. 進入後台管理
```

### 2. 自定義建議
如果想要調整顏色，修改 `/static/css/custom.css` 中的：
```css
:root {
    --primary-color: #4e73df;  /* 主色調 */
    --success-color: #1cc88a;  /* 成功色 */
    /* ... */
}
```

### 3. 商品圖片
- 建議尺寸：800x600 px
- 格式：JPG/PNG
- 來源：Unsplash 或實際商品圖
